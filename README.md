## What Is an AI Agent?

An AI agent is an LLM that can:
- **Reason** — think through multi-step problems
- **Use tools** — search, execute code, read files, call APIs
- **Remember** — maintain context across steps
- **Act** — take actions and observe results

## 5 Agents to Build

### 1. Research Agent
Takes a research question, searches the web, reads sources,
synthesizes into a structured report with citations.

### 2. Data Analysis Agent
Loads a dataset, runs EDA automatically, generates visualizations,
writes insights, answers follow-up questions.

### 3. Coding Agent
Reads a feature request, reads codebase, writes implementation,
runs tests, fixes failures, commits to git.

### 4. Finance Agent
Fetches market data, runs analysis (returns, volatility, Sharpe),
generates trade signals, creates report with charts.

### 5. Customer Support Agent
Reads message, searches knowledge base, responds or escalates,
updates CRM.

## Agent Frameworks

| Framework | Best For |
|---|---|
| LangGraph | Complex state machines, branching |
| CrewAI | Multi-agent teams with roles |
| AutoGen | Multi-agent conversations |
| Raw API | Simple ReAct agents |
